# Gest-o-de-League-of-Legends
A Riot Games rastreia milhões de itens, gastos classificações. Como organizar esses  dados de forma integra e eficiente?  Pensando nisso, me veio a ideia de modelar um banco de dados que garanta a  integridade, rastreabilidade financeira (RP - moeda do jogo) e facilite análises  complexas sobre o jogador.
